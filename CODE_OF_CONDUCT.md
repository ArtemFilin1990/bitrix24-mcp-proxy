# Code of Conduct

Мы придерживаемся принципов, изложенных в [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

- Будьте уважительны и доброжелательны.
- Предоставляйте конструктивный фидбек.
- Не допускайте дискриминации или домогательств.

Нарушения можно сообщать владельцу репозитория через GitHub Issues или напрямую на электронную почту, указанную в профиле владельца.
