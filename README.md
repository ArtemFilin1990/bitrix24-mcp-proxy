# Bitrix24 MCP Proxy for Vercel

API-proxy для MCP-сервера Bitrix24. Совместим с ChatGPT Custom Actions.

## Endpoints

### /servers
Возвращает список MCP-серверов.

### /mcp
Проксирует MCP-запросы в Bitrix24 MCP: https://mcp-dev.bitrix24.tech/mcp

## Deploy
1. Push в GitHub  
2. Import в Vercel  
3. Deploy  
4. Использовать URL:  
   https://PROJECT.vercel.app/servers
